# Support

## Start Here

Check the README troubleshooting section and run:

```bash
agent-knock-knock doctor
```

Search the existing GitHub issues before opening a new one. AKK supports Codex, Claude Code, and Cursor through ACPX. Native tmux control currently supports Codex and Claude Code; Cursor tmux support is tracked in issue #42.

## Ask for Help

- Use the [bug report form](https://github.com/scotthuang/agent-knock-knock/issues/new?template=bug_report.yml) for reproducible failures.
- Use the [feature request form](https://github.com/scotthuang/agent-knock-knock/issues/new?template=feature_request.yml) for proposed behavior.
- For a usage question, open a GitHub issue with a concise `[Support]` title, the relevant versions, and the expected workflow.

Share only the minimum diagnostics needed to understand the problem. Redact local paths when appropriate, and never include gateway tokens, API keys, passwords, proxy credentials, private prompts, raw callback payloads, or unreviewed task logs.

## Security

Do not use a public support issue for vulnerabilities or sensitive findings. Follow the private reporting instructions in [SECURITY.md](SECURITY.md).
